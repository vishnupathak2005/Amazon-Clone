# Amazon Homepage Clone

A frontend clone of the Amazon homepage built using **HTML5** and **CSS3**. This project was created to strengthen my understanding of web layout, Flexbox, responsive design, and UI development by recreating one of the world's most popular e-commerce websites.

## Features

- Amazon-inspired homepage layout
- Responsive navigation bar
- Search bar with category dropdown
- Product category cards
- Hero banner section
- Footer with multiple information sections
- Clean and organized HTML & CSS structure
- Font Awesome icons integration

## Technologies Used

- HTML5
- CSS3
- Font Awesome
- Google Fonts
